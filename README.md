# Doon

Doon은 사용자의 컴퓨터 화면에서 작업 과정을 보여주고, 단계별 승인을 받으며 귀찮은 디지털 작업을 실제로 끝내는 개인용 AI 컴퓨터 오퍼레이터입니다.

## Current Direction

- macOS 데스크톱 앱을 첫 실행 환경으로 삼습니다.
- 초기 실행은 GUI 조작에 집중합니다.
- 작업을 단계별 결과물로 나누고, 각 단계에서 사용자 승인과 개입을 지원합니다.
- 데스크톱 앱이 안정화된 뒤 설치용 홈페이지를 제공합니다.
